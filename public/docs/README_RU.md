## Три Фазы

| Фаза | Цель | Ключевые возможности | Примеры навыков | Статус |
| --- | --- | --- | --- | --- |
| **Start** | Инициализация проекта | Создание структур папок, готовых к продакшену, инициализация фреймворков, применение лучших практик с первого коммита. | `start-nextjs`... | Запланировано |
| **Middle** | Непрерывное улучшение | Улучшение UI/UX, повышение безопасности, производительности, очистка логики. | TBD | Запланировано |
| **End** | Аудит и рефакторинг | Полный анализ архитектуры. План по этапам выполняется **только с явного одобрения**. | `refactor-project` | ✅ |

## Доступные навыки

| Навык | Описание |
| --- | --- |
| **[end](../../skills/end)** | **`refactor-project`** — Понимает ваш проект от начала до конца. Предоставляет четкий диагноз с конкретными ссылками. Рекомендует правильное направление архитектуры. Вы сохраняете полный контроль. |

## Установка

```bash
# Добавить текущий навык (End / refactor-project)
npx skills add bastndev/skills --skill end
```

Будущие навыки будут устанавливаться так же:

```bash
npx skills add bastndev/skills --skill start-nextjs
```

## Как работает навык End

1. **Только анализ** — Картографирует точки входа, понимает структуру. **Никаких модификаций.**
2. **Структурированный отчет** — Находки делятся на Ошибки, Риски, Возможности и Технический долг.
3. **Индивидуальный план** — Фазы упорядочены по их реальной ценности для проекта.
4. **Пошаговое выполнение** — Выполняет _одну_ фазу после вашего разрешения и останавливается.
5. **Безопасность превыше всего** — Никогда не создает тесты, если их не было. Не добавляет зависимости без разрешения.

Полная спецификация: [skills/end/SKILL.md](../../skills/end/SKILL.md).

## Дорожная карта

- Семейство **Start**: Генераторы проектов в одну команду.
- Семейство **Middle**: Специалисты по целенаправленным улучшениям.
- Расширение **End** и поддержка новых языков.
