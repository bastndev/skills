| المرحلة | الغرض | القدرات الرئيسية | أمثلة | الحالة |

| المهارة | الوصف |

| **[end](../../skills/end)** | **`refactor-project`** — يفهم مشروعك من البداية إلى النهاية. يقدم تشخيصًا واضحًا مع مراجع ملموسة. يوصي بالاتجاه المعماري الصحيح. أنت تبقى في السيطرة الكاملة. |

## التثبيت

```bash

npx skills add bastndev/skills --skill end
```

المهارات المستقبلية سيتم تثبيتها بنفس الطريقة:

```bash
npx skills add bastndev/skills --skill start-nextjs
```

## كيف تعمل مهارة End

1. **التحليل فقط** — يحدد نقاط الدخول، يفهم البنية. **بدون تعديلات.**
2. **تقرير منظم** — تُفصل النتائج إلى أخطاء، مخاطر، فرص، وديون تقنية.
3. **خطة مخصصة** — تُطلب المراحل حسب القيمة الحقيقية لمشروعك.
4. **تنفيذ خطوة بخطوة** — بعد تفويضك، ينفذ مرحلة _واحدة_ ويتوقف.
5. **الأمان أولاً** — لا ينشئ اختبارات أبدًا إذا لم تكن موجودة. لا يضيف تبعيات بدون إذن.

شاهد المواصفات الكاملة في [skills/end/SKILL.md](../../skills/end/SKILL.md).

## خارطة الطريق

- عائلة **Start**: مولدات مشاريع بأمر واحد.
- عائلة **Middle**: متخصصون في التحسينات المركزة.
- توسيع قدرات **End** ودعم لغات إضافية.
