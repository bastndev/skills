## التثبيت

كل مهارة تُثبّت بنفس الطريقة — اختر ما تحتاجه:

```bash
npx skills add bastndev/skills --skill start-package   # Start  — scaffold a TS npm package
npx skills add bastndev/skills --skill start-astro     # Start  — scaffold an Astro project
npx skills add bastndev/skills --skill skrapi          # Start  — study & document any codebase
npx skills add bastndev/skills --skill middle          # Middle — score & improve one dimension
npx skills add bastndev/skills --skill end             # End    — audit & safely refactor
npx skills add bastndev/skills --skill l10n-sync       # Utility — sync localized assets safely
```

## كيف تعمل مهارة End

`end` هي المهارة الأ成熟 في المجموعة. إليك سير عملها من البداية إلى النهاية:

1. **التحليل أولاً** — يُحدد نقاط الدخول ويتفهّم المشروع. **لا تُعدّل أي ملفات.**
2. **تقرير منظم** — نتائج واضحة مصنّفة إلى أخطاء (مع الخطورة)، دين/مخاطر، واقتراحات، بالإضافة إلى نظرة عامة على الصحة، وتوصية معمارية، وخطة مرتبة — كلها مدعومة بمراجع محددة للملف + السطر.
3. **أنت تُصرّح لكل مرحلة** — ينفّذ **مرحلة واحدة فقط** في كل مرة. بعد كل مرحلة تحصل على ملخص دقيق للتغييرات، والتحقق الذي أُجري، وقائمة المراحل المتبقية.
4. **تحكم كامل وأمان** — لا يُنشئ اختبارات إذا لم يكن المشروع بها. لا يُضيف تبعيات أو يُغيّر مدير الحزم بدون إذن. يحترم عملك غير المُ Commit ويحافظ دائمًا على السلوك الحالي إلا إذا كان بإصلاح خطأ مبرر.

للحصول على سير العمل الكامل، وتنسيقات التقارير الدقيقة (بما في ذلك كتل الإغلاق المطلوبة)، وقواعد القرار المعماري، وجميع ضمانات الأمان، اقرأ توثيق المهارة المُخصّص:

→ **[End – Refactor Project](./skills/end/README.md)**

المواصفة الداخلية الكاملة موجودة في [skills/end/SKILL.md](./skills/end/SKILL.md).

## خارطة الطريق

- **Start** — `start-package` (إنشاء هيكل) و`start-astro` (إنشاء هيكل Astro) و`skrapi` (دراسة كود موجود) متوفرة اليوم؛ مُنشئات `start-*` إضافية قادمة.
- **Middle** — `middle` متوفرة اليوم (نظرة عامة على الصحة + ستة محسّنات مرقّمة)؛ أدوات أعمق لكل بُعد مخطّطة.
- **End** — `end` متوفرة اليوم؛ بيئات تشغيل إضافية، وأنماط إعادة هيكلة متخصصة، وأدوات مساعدة مخطّطة.
- **Utility** — `l10n-sync` متوفرة اليوم للتوطين المتّقَن وموفر الرموز.

كل مهارة تأتي مع توثيق مُخصّص بها (مثل [End – Refactor Project](./skills/end/README.md) الحالي).
