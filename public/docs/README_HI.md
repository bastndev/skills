## स्थापना

```bash
# वर्तमान कौशल जोड़ें (End / refactor-project)
npx skills add bastndev/skills --skill end
```

भविष्य के कौशल भी इसी तरह स्थापित किए जा सकेंगे:

## End कौशल कैसे काम करता है

1. **केवल विश्लेषण** — प्रवेश बिंदुओं को मैप करता है, संरचना को समझता है। **शून्य संशोधन।**
2. **संरचित रिपोर्ट** — निष्कर्षों को बग्स, जोखिम, अवसर और तकनीकी ऋण में अलग किया गया है।
3. **अनुकूलित योजना** — आपके प्रोजेक्ट के वास्तविक मूल्य के अनुसार चरणों को आदेशित किया जाता है।
4. **चरण-दर-चरण निष्पादन** — आपके प्राधिकरण के बाद, यह _एक_ चरण निष्पादित करता है और प्रतीक्षा करता है।
5. **सुरक्षा पहले** — अनुमति के बिना निर्भरता कभी नहीं जोड़ता।

पूरा विवरण [skills/end/SKILL.md](../../skills/end/SKILL.md) पर देखें।
