## Cài đặt

```bash
# Thêm skill hiện tại (End / refactor-project)
npx skills add bastndev/skills --skill end
```

Các skill trong tương lai sẽ được cài đặt theo cách tương tự:

```bash
npx skills add bastndev/skills --skill start-nextjs
```

## Skill End hoạt động như thế nào

1. **Chỉ phân tích** — Lập bản đồ các điểm truy cập, hiểu cấu trúc, đánh giá code liên quan. **Không sửa đổi.**
2. **Báo cáo có cấu trúc** — Các phát hiện được chia thành Bugs, Rủi ro, Cơ hội Tái cấu trúc và Nợ Kỹ thuật.
3. **Kế hoạch tùy chỉnh** — Các giai đoạn được sắp xếp theo giá trị thực đối với dự án của bạn.
4. **Thực thi từng bước** — Sau khi bạn cho phép, nó thực hiện _một_ giai đoạn rồi dừng lại.
5. **An toàn là trên hết** — Không bao giờ tạo bài kiểm tra nếu không có sẵn. Không thêm phụ thuộc mà không có sự cho phép.

Xem thông số kỹ thuật đầy đủ tại [skills/end/SKILL.md](../../skills/end/SKILL.md).
