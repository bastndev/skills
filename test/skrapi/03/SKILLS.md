## Skills (.agents/skills/)

### 1. **skrapi**
Analiza la arquitectura de cualquier proyecto de software — web, móvil, extensión de navegador, o paquete npm — y produce una carpeta de archivos Markdown enfocados sobre arquitectura y dependencias. Cubre proyectos en cualquier stack JS/TS (Next.js, Astro, React Native, Expo, Lynx JS, Vite, etc.) o otros lenguajes si es necesario.

**Triggers**: cuando el usuario quiere entender o documentar un codebase existente; estudiar la estructura de otro proyecto para inspiración; preguntar qué paquete maneja algo específico (routing, state management, formularios, i18n, etc.).

**Referencias** (4 archivos):
- `references/web.md` — Qué buscar en proyectos web (Next.js, Astro, Vite, Remix, Nuxt, etc.)
- `references/mobile.md` — Qué buscar en apps móviles (React Native, Expo, Lynx JS)
- `references/extension.md` — Qué buscar en extensiones de navegador
- `references/library.md` — Qué buscar en paquetes npm/librerías/CLIs
