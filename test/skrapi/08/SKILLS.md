## Skills (.agents/skills/)

### 1. **skrapi**
Analyzes any software project (web, mobile, extension, npm library) across JS/TS stacks and produces a `SKRAPI/` folder of focused Markdown docs covering architecture, packages, and AI-tooling context.
**Triggers**: when the user wants to understand, document, or compare a codebase's structure, packages, or skill setup.

### 2. **improve-codebase-architecture**
Audits and improves the structural architecture of a codebase — includes interface design, deepening analysis, language guidance, and HTML report generation (5 supporting files).
**Triggers**: when the user asks for an architectural review, refactor suggestions, or wants a structured improvement plan.

### 3. **gpt-image-2**
Generates images using OpenAI's GPT-image-2 model.
**Triggers**: when the user asks to generate, create, or produce an image.

### 4. **auth0-quickstart**
Integrates Auth0 authentication into a project — covers setup, configuration, and quickstart patterns (includes a `references/` folder).
**Triggers**: when the user wants to add Auth0 login, authentication, or identity management.

## Skills (.claude/skills/)

### 5. **vercel-react-best-practices**
React and Next.js performance optimization guidelines from Vercel Engineering — covers component patterns, data fetching, bundle optimization, and rendering best practices (includes a `rules/` folder with detailed guidelines).
**Triggers**: when writing, reviewing, or refactoring React/Next.js code, or when addressing performance, bundle size, or data fetching concerns.

### 6. **bun**
Guides usage of Bun as a runtime, package manager, and test runner — install commands, script execution, and Bun-specific APIs.
**Triggers**: when the user asks about running scripts, installing packages, or using Bun features in the project.
