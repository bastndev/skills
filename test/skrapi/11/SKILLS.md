## Skills (.agents/skills/ y .claude/skills/)

### 1. **auth0-quickstart**
Configuración rápida de autenticación con Auth0. Detecta el stack del proyecto y enruta al workflow de SDK correcto.
**Triggers**: Cuando se añade autenticación o login a cualquier app.

### 2. **bun**
Herramienta unificada para construir, testear y desplegar aplicaciones JS/TS. Maneja scripts, dependencias, bundling y testing.
**Triggers**: Al ejecutar scripts, gestionar dependencias, bundlear código o testear con Bun.

### 3. **gpt-image-2**
Generación y edición de imágenes con OpenAI GPT Image 2 en RunComfy. Fortalezas: texto embebido, logos, tipografía multilingüe.
**Triggers**: "gpt image 2", "gpt-image-2", "ChatGPT Images 2", o peticiones explícitas de generación/edición con este modelo.

### 4. **improve-codebase-architecture**
Analiza oportunidades de mejora arquitectónica en el codebase, informado por el lenguaje del dominio y decisiones en ADRs.
**Triggers**: Cuando el usuario quiere mejorar la arquitectura, encontrar refactorizaciones, o consolidar módulos acoplados.

### 5. **skrapi**
Analizador de arquitectura para proyectos de software — web, móvil, extensión de navegador, o paquetes npm — produce documentación Markdown enfocada.
**Triggers**: "analiza este proyecto", "documenta la arquitectura", "qué packages usa", o cualquier petición de entendimiento/ documentación de codebase.

### 6. **vercel-react-best-practices**
Directrices de optimización de rendimiento de React/Next.js de Vercel Engineering. Patrones óptimos para componentes, data fetching, y bundle optimization.
**Triggers**: Al escribir, revisar o refactorear código React/Next.js para asegurar patrones de rendimiento óptimos.
