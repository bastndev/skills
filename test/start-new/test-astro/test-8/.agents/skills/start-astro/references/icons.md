# Icons — Lucide + custom brand/theme SVGs

Two sources, no React anywhere:

- **Lucide** (`@lucide/astro`) for standard line icons. Install with `bun add @lucide/astro` (the only runtime dep this skill adds). Use directly: `import { Search } from '@lucide/astro'` → `<Search />`.
- **Custom icons** as raw `.svg` files under `src/assets/icons/`, imported as components (Astro 7 renders `*.svg` imports inline, so they inherit `currentColor`). Brand logos live here because **Lucide ships no brand marks**.

Write the files below exactly. Filenames are lowercase `kebab-case`. All use `currentColor` so CSS controls color (and they follow the theme); size them with CSS, e.g. `.icon { width: 24px; height: 24px; color: var(--color-nav-text); }`.

Usage:
```astro
---
import Twitter from '@/assets/icons/social/twitter.svg';
import { Search } from '@lucide/astro';
---
<Twitter class="icon" />
<Search class="icon" />
```

> The theme toggle (`Header.astro`) imports `theme/sun.svg` + `theme/moon.svg` and renders them as `<Sun id="icon-sun" aria-hidden="true" />` / `<Moon id="icon-moon" aria-hidden="true" />`; their show/hide swap is in `global.css` (see `references/global-css.md`).
